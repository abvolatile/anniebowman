Version Control

Version Control

Version control is a useful skill to have at any company where multiple revisions need to be made on a project and where there are many people working on the same thing at once. When writing computer programs, everything is dependent on the tiniest detail and one small error could create total chaos. With version control, every change is tracked and each version is saved so team members can more easily fix bugs and find the best and most efficient way to implement their program.


I have learned the theories and concepts behind version control from my education at The Tech Academy, and specifically using git and GitHub. I have also supplemented this knowledge with other resources like Udacity to make sure I have a solid grasp of it's use.

I know this!
  • Theory and Concepts
  • Command-line git
  • GitHub

.NET Framework

The .NET Framework is Microsoft's collection of tools and infrastructure that allow developers to use several different languages and compile them all into a platform-neutral intermediate language so that it can run within the CLR on any Windows machine.
